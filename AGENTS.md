## Scope

## Skill Sync

When changing config that affects the advanced swap workflow, update the skill in the same change.

This includes `config.json` and `skills/advanced-swap-orders/scripts/skill.config.json`.

Keep `skills/advanced-swap-orders/SKILL.md` aligned with those config changes, and update any directly referenced skill docs or assets when the supported chains, addresses, endpoints, parameters, or operator workflow change.
